3.6、客户端

获取 rbd pool image 的客户端列表。具体的执行操作流程如下:

  1. ceph --conf %s --user admin -f json mon stat : 获取 mon 节点名称列表;
  2. ceph --conf %s --user admin -f json tell mon.%s sessions : 获取 mon 节点上的 session(client) 列表信息;
  3. ceph --conf %s --user admin -f json osd pool ls detail : 获取 rbd 类型的数据池;
  4. rbd --conf %s --id admin list --format json --pool %s : 遍历 rbd 类型的数据池其中的映像列表;
  5. rbd --conf %s --id admin info --format json --pool %s --image %s : 获取 rbd 类型的数据池映像的块名前缀信息;
  6. rados --conf %s --id admin --format json --pool %s listwatchers rbd_header.%s : 获取 rbd 池中监听对应块名对象的客户端;
