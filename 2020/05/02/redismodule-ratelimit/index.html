RateLimit 是一款基于 Go的限速库 golang.org/x/time/rate （基于 令牌桶 ） 实现的针对于 key 的限速模块，该模块并非直接拦截 Redis 中关于特定 key 的操作指令，而是每次在需要执行操作指令之前，先发送一个判断命令（该模块提供的特殊命令），通过这种方式来实现限速的目的。
