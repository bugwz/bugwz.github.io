一、iTerm2的Triggers功能

关于iTerm2的 Triggers 功能，官方的介绍是：

A trigger is an action that is performed when text matching some regular expression is received in a terminal session.

即：触发器是在终端会话中收到与某个正则表达式匹配的文本时执行的动作。
